## Instalação da front

1-) clonar projeto https://github.com/phelipperibeiro/laravel_angular_front.git

2-) rodar o comando: composer install (gerenciador de dependencia do php, composer.json)

3-) rodar o comnado: npm install (gerenciador de dependencia do nodeJS, package.json)

4-) rodar o comando: php artisan key:generate 

5-) (caso nao tenha instalado) instalar bower com o comando: npm install -g bower (-g global)

6-) rodar o comando: bower install (gerenciador de dependecia JS, bower.json)
    obs: lembrando que o arquivo (.bowerrc) contem o path do diretorio que guarda as libs
